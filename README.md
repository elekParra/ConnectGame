# 🔢 Conecta PRO

Un juego de lógica donde debes conectar los números en orden dentro de una cuadrícula.

##Como jugar

- Empieza en el número 1
- Conecta los números en orden ascendente
- Solo puedes moverte a casillas adyacentes
- Completa el recorrido sin errores

## Características

- Generación procedural de niveles
- 7–9 números por nivel
- Dificultad progresiva
- Controles táctiles y ratón
- Optimizado para móvil y desktop
- Juego 100% en HTML, CSS y JavaScript (Canvas)

##Objetivo

Entrenar lógica, memoria y planificación mientras completas rutas cada vez más complejas.

## Jugar online

Alojado con GitHub Pages, podrás jugar directamente desde el navegador sin instalación.

## Tecnologías

- HTML5 Canvas
- JavaScript (Vanilla)
- CSS moderno responsive

## Compatibilidad

Funciona en:
- Móviles 📱
- Tablets 📲
- PC 💻
